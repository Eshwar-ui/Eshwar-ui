## 〔 Design Philosophy 〕

<br />

<div align="center">

<img src="https://img.shields.io/badge/%E2%80%9CIf%20it%20feels%20smooth%2C%20simple%2C%20and%20intentional%20%E2%80%94%20it's%20a%20great%20product.%E2%80%9D-6C3BFF?style=for-the-badge&labelColor=0d1117" />
